= 닉네임 변경 API
:toc: left
:sectnums:

닉네임 변경 API는 로그인된 사용자가 자신의 닉네임을 새로운 값으로 변경합니다.
닉네임은 중복될 수 없으며, 이미 존재하는 닉네임으로 변경 시 오류가 발생합니다.

유효한 토큰과 새로운 닉네임으로 요청 시, 닉네임이 성공적으로 변경됩니다.
