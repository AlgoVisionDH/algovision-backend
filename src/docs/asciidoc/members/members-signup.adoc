== 🧾 개요
회원가입 API는 신규 사용자를 등록하는 엔드포인트입니다.
입력값 검증(이메일 중복, 닉네임 중복, 비밀번호 불일치 등)을 수행하며,
정상 등록 시 새 회원의 정보를 반환합니다.
