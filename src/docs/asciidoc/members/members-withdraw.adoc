= 회원 탈퇴 API
:toc: left
:sectnums:

회원 탈퇴 API는 현재 로그인한 사용자의 계정을 **소프트 삭제(soft delete)** 처리합니다.
요청 시 서버는 해당 회원의 Refresh Token을 삭제하고, Access Token을 블랙리스트에 등록합니다.
